# Курс верстки лендинга для образовательного канала ITVDN
## Перед стартом проекта необходимо

* Убедится что установлена node и npm. Для этого достаточно написать в терминале
> nide -v

Если вы видете версию например v6.11.0 то всё ок

>npm -v 

Если вы видете версию например 3.10.10 то все ок



* Убедиться что установлен gulp 4 версии, так как будет меньше - работать не будет. 
Если он не установлен тогда ставим:
 >npm install gulpjs/gulp-cli#4.0 -g
 
 
 
## Инструкция для старта проекта:
* Склонировать данный репозиторий 
>git clone https://github.com/Dibaz0l/first-landing.git

* Запустить команду npm install в терминале. Данная команда становит все пакеты, которые указаны в файле packege.json, а также все их зависимости
